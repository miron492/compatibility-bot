# Бот для проверки совместимости пар

Telegram-бот для оценки совместимости между двумя людьми.

## Особенности:
- Каждый заполняет анкету один раз
- Уникальный код для каждой анкеты
- Проверка совместимости по коду
- Подробные результаты с прогнозами

## Как использовать:
1. Запустить бота
2. Пройти анкету (12 вопросов)
3. Получить код своей анкеты
4. Поделиться кодом с другим человеком
5. Проверить совместимость по коду

## Технологии:
- Python 3.13
- Aiogram 3.24
- SQLite
